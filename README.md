## Run Locally

Clone the project

```bash
  git clone https://github.com/NiTrO0FuN/poker-manager
```

Go to the project directory

```bash
  cd poker-manager
```

Install dependencies

```bash
  npm install
```

Start the dev server

```bash
  npm run dev
```


## Tech Stack

**Client:** [Vue.js](https://vuejs.org/), [Vite](https://vitejs.dev/), [PrimeVue](https://primevue.org/)
